BlinkList Commands
-------------------------------------------------------------------------
1. Add
add DD MMM HH:MM Description

2. Edit
edit # DD MMM HH:MM Description

3. Delete
delete #

4. Show
show today
show tomorrow
show this month

5. Search
search Keyword

6. Undo 
undo

7. Done
done #

* # refers to the number on your displayed list

-------------------------------end of all commands-----------------------
